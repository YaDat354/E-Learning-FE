# E-Learning FE

## Cách chạy project

1. Cài dependencies:

```bash
npm install
```

2. Chạy môi trường development:

```bash
npm run dev
```

3. Mở trình duyệt tại địa chỉ hiển thị trong terminal (thường là `http://localhost:5173`).

## Các lệnh khác

Build production:

```bash
npm run build
```

Preview bản build:

```bash
npm run preview
```
